Positionierung

static

Positionierung im Dokumentfluss.

div	{
	position: static;
}

Positionierung

relative

Positionierung relativ zur normalen Position des Elements.

div	{
	position: relative;
	top: -40px;
	right: 25px;
}

Positionierung

absolute

Positionierung relativ zum nächsten Elternelment, welches nicht static ist.

.container	{
	position: relative;
}

.content	{
	position: absolute;
	top: -40px;
	right: 25px;
}

Positionierung

fixed

Positionierung relativ zum Browserfenster.

div	{
	position: fixed;
	top: 40px;
	left: 25px;
}

Positionierung

absolute & fixed
