Wiederholung | Einheit 5 | Positionierung

relative

Positionierung relativ zur normalen Position des Elements.

div	{
	position: relative;
	top: -40px;
	right: 25px;
}

Wiederholung | Einheit 5 | Positionierung

absolute

Positionierung relativ zum nächsten Elternelment, welches nicht static ist.

.container	{
	position: relative;
}

.content	{
	position: absolute;
	top: -40px;
	right: 25px;
}

Wiederholung | Einheit 5 | Positionierung
